# Скрипты для парсинга автомобилей с brauncar.com

## Описание

Эти скрипты предназначены для парсинга информации об автомобилях и скачивания их изображений с сайта brauncar.com.

## Скрипты

### 1. Базовый скрипт (`parse-cars.js`)

Простой скрипт, который пытается скачать изображения по известным URL-паттернам.

**Использование:**
```bash
npm run parse-cars
```

**Что делает:**
- Пытается скачать изображения для каждого автомобиля по различным URL-паттернам
- Сохраняет изображения в `public/cars/`
- Создает файл `public/cars/images.json` с результатами

### 2. Продвинутый скрипт (`parse-cars-advanced.js`)

Более продвинутый скрипт, использующий Puppeteer для парсинга JavaScript-контента сайта.

**Требования:**
```bash
npm install puppeteer
```

**Использование:**
```bash
node scripts/parse-cars-advanced.js
```

**Что делает:**
- Открывает сайт brauncar.com в браузере
- Парсит все карточки автомобилей
- Извлекает все изображения для каждого автомобиля
- Скачивает изображения локально
- Сохраняет результаты в JSON

## Структура файлов

После выполнения скриптов структура будет следующей:

```
public/
  cars/
    mg3-blue.jpg
    mg3-blue-1.jpg
    mg3-black.jpg
    ...
    images.json
```

## Формат images.json

```json
{
  "mg3-blue": [
    "/cars/mg3-blue.jpg",
    "/cars/mg3-blue-1.jpg"
  ],
  "mg3-black": [
    "/cars/mg3-black.jpg"
  ]
}
```

## Ручная загрузка изображений

Если скрипты не смогли автоматически скачать изображения:

1. Откройте сайт brauncar.com в браузере
2. Откройте инструменты разработчика (F12)
3. Найдите изображения автомобилей в Network или Elements
4. Скачайте изображения вручную
5. Сохраните их в `public/cars/` с именами вида `{car-id}.jpg`
6. Обновите `images.json` вручную

## Интеграция с компонентом

Компонент `RentPage.tsx` автоматически использует локальные изображения, если они доступны:

1. Сначала пытается загрузить из `/cars/{car-id}.jpg`
2. Если не найдено, использует удаленный URL
3. Если и это не работает, использует fallback изображение

## Примечания

- Скрипты добавляют задержки между запросами, чтобы не перегружать сервер
- Если изображения не найдены автоматически, их можно загрузить вручную
- Убедитесь, что папка `public/cars/` существует перед запуском скриптов
